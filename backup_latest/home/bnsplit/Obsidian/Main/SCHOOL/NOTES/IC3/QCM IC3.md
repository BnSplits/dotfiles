# QCM : IC3 (Windows et Word)

## Windows 

##### 1. Quel est le rôle du menu Démarrer dans Windows ?
a) Éteindre l'ordinateur
b) Accéder rapidement aux applications et paramètres
c) Désinstaller les logiciels
d) Modifier les fichiers système

_Réponse : b) Accéder rapidement aux applications et paramètres_

##### 2. Quel raccourci clavier permet d'ouvrir l'Explorateur de fichiers ?
a) Ctrl + E
b) Windows + E
c) Alt + Tab
d) Ctrl + Shift + E

_Réponse : b) Windows + E_

##### 3. Que signifie "Ctrl + Z" sous Windows ?
a) Copier
b) Annuler la dernière action
c) Enregistrer
d) Fermer une fenêtre

_Réponse : b) Annuler la dernière action_

##### 4. Où peut-on changer le fond d’écran dans Windows ?
a) Explorateur de fichiers
b) Panneau de configuration ou Paramètres
c) Invite de commandes
d) Microsoft Store

_Réponse : b) Panneau de configuration ou Paramètres_

##### 5. Quel est le navigateur par défaut de Windows 11 ?
a) Internet Explorer
b) Mozilla Firefox
c) Google Chrome
d) Microsoft Edge

_Réponse : d) Microsoft Edge_

##### 6. Comment créer un dossier dans Windows ?
a) Clic droit > Nouveau > Dossier
b) Ctrl + D
c) Ctrl + N
d) Fichier > Nouveau

_Réponse : a) Clic droit > Nouveau > Dossier_

##### 7. Quel est l’extension par défaut des fichiers Word dans Windows ?
a) .txt
b) .doc
c) .docx
d) .pdf

_Réponse : c) .docx_

##### 8. Quelle combinaison de touches permet de basculer entre les fenêtres ouvertes ?
a) Ctrl + Tab
b) Alt + Tab
c) Windows + Tab
d) Shift + Tab

_Réponse : b) Alt + Tab_

##### 9. Que fait la commande "Ctrl + P" ?
a) Colle un texte
b) Imprime un document
c) Ouvre un nouveau fichier
d) Annule une action

_Réponse : b) Imprime un document_

##### 10. Où peut-on désinstaller un logiciel dans Windows ?
a) Explorateur de fichiers
b) Gestionnaire de périphériques
c) Applications et fonctionnalités dans Paramètres
d) Centre de sécurité Windows

_Réponse : c) Applications et fonctionnalités dans Paramètres_

## Word
##### 11. Que permet de faire le raccourci "Ctrl + N" dans Word ?
a) Ouvrir un fichier existant
b) Créer un nouveau document
c) Enregistrer le document
d) Copier le contenu

_Réponse : b) Créer un nouveau document_

##### 12. Comment insérer une image dans un document Word ?
a) Onglet Accueil > Image
b) Onglet Insertion > Image
c) Onglet Affichage > Image
d) Onglet Références > Image

_Réponse : b) Onglet Insertion > Image_

##### 13. Quelle est l’utilité de l’outil "Vérification orthographique" dans Word ?
a) Supprimer les fautes automatiquement
b) Identifier les fautes d’orthographe et de grammaire
c) Formater le texte
d) Convertir un document en PDF

_Réponse : b) Identifier les fautes d’orthographe et de grammaire_

##### 14. Que fait la combinaison "Ctrl + S" dans Word ?
a) Sauvegarde le document
b) Supprime le texte sélectionné
c) Crée un nouveau fichier
d) Ouvre une recherche

_Réponse : a) Sauvegarde le document_

##### 15. Comment sélectionner tout le texte d’un document Word ?
a) Ctrl + S
b) Ctrl + T
c) Ctrl + A
d) Ctrl + E

_Réponse : c) Ctrl + A_

##### 16. Quel est le rôle de l'onglet "Mise en page" dans Word ?
a) Changer les marges, l'orientation et le format du document
b) Modifier la police et les couleurs
c) Ajouter des graphiques
d) Insérer des notes de bas de page

_Réponse : a) Changer les marges, l'orientation et le format du document_

##### 17. Comment insérer un tableau dans Word ?
a) Onglet Références > Tableau
b) Onglet Insertion > Tableau
c) Onglet Accueil > Tableau
d) Onglet Affichage > Tableau

_Réponse : b) Onglet Insertion > Tableau_

##### 18. Que fait le raccourci "Ctrl + C" dans Word ?
a) Coupe le texte sélectionné
b) Copie le texte sélectionné
c) Colle le texte copié
d) Annule une action

_Réponse : b) Copie le texte sélectionné_

##### 19. Que permet de faire l'outil "Thème" dans Word ?
a) Modifier le fond du document
b) Ajouter des effets et des couleurs prédéfinies
c) Créer une table des matières
d) Convertir le fichier en PDF

_Réponse : b) Ajouter des effets et des couleurs prédéfinies_

##### 20. Où se trouve l'option pour ajouter une numérotation des pages ?
a) Onglet Accueil > Numérotation
b) Onglet Références > Pages
c) Onglet Insertion > Numéro de page
d) Onglet Affichage > Numérotation

_Réponse : c) Onglet Insertion > Numéro de page_

##### 21. Quel format de fichier garantit que la mise en page d'un document Word reste intacte ?
a) .txt
b) .pdf
c) .doc
d) .html

_Réponse : b) .pdf_

##### 22. Quelle fonctionnalité de Word est utile pour collaborer à distance ?
a) Partage via OneDrive
b) Mode Hors connexion
c) Options d'impression
d) Onglet Mise en page

_Réponse : a) Partage via OneDrive_

##### 23. Comment insérer un en-tête dans Word ?
a) Onglet Affichage > En-tête
b) Onglet Références > En-tête
c) Onglet Insertion > En-tête et pied de page
d) Onglet Accueil > En-tête

_Réponse : c) Onglet Insertion > En-tête et pied de page_

##### 24. Quelle option permet de trouver un mot spécifique dans un document Word ?
a) Onglet Références > Recherche
b) Onglet Accueil > Rechercher
c) Onglet Insertion > Rechercher
d) Onglet Mise en page > Recherche

_Réponse : b) Onglet Accueil > Rechercher_

##### 25. Que fait la commande "Ctrl + P" dans Word ?
a) Enregistre le fichier
b) Colle un contenu
c) Imprime le document
d) Supprime un paragraphe

_Réponse : c) Imprime le document_

##### 26. Comment modifier l’interligne dans Word ?
a) Onglet Accueil > Paragraphe > Espacement
b) Onglet Insertion > Paragraphe
c) Onglet Références > Interligne
d) Onglet Mise en page > Espacement

_Réponse : a) Onglet Accueil > Paragraphe > Espacement_

##### 27. Quelle option permet de diviser un document en plusieurs colonnes ?
a) Onglet Mise en page > Colonnes
b) Onglet Insertion > Colonnes
c) Onglet Accueil > Colonnes
d) Onglet Références > Colonnes

_Réponse : a) Onglet Mise en page > Colonnes_

##### 28. Comment insérer un saut de page dans Word ?
a) Ctrl + P
b) Ctrl + Entrée
c) Ctrl + Z
d) Ctrl + F

_Réponse : b) Ctrl + Entrée_

##### 29. Quel outil permet de créer automatiquement une table des matières ?
a) Onglet Références > Table des matières
b) Onglet Accueil > Table
c) Onglet Insertion > Références
d) Onglet Mise en page > Table des matières

_Réponse : a) Onglet Références > Table des matières_

##### 30. Où peut-on modifier l’orientation des pages dans Word ?
a) Onglet Références
b) Onglet Mise en page
c) Onglet Insertion
d) Onglet Accueil

_Réponse : b) Onglet Mise en page_
